# Data Structures & Algorithms in Java

My implementation of the data structures and algorithms covered in **COS212: Data Structures and Algorithms** at the **University of Pretoria**.

## Why this repository?

Data structures are fairly easy to understand as humans. We naturally think in terms of trees, lists, networks, and relationships. The real challenge begins when you have to teach a computer those same concepts.

I can definitely attest to that struggle.

This repository is my way of working through that process by implementing data structures and algorithms from scratch in Java. The goal isn't just to get them working, but to understand *why* they work, the trade-offs behind their design, and how they behave under different situations.

Along the way, I'll be implementing:

* Fundamental data structures
* Classic algorithms
* Variations and optimisations
* Small experiments to deepen my understanding

## Why another DSA repository?

There are already countless DSA repositories online, and I'm well aware of that.

This one exists primarily as a learning journal. Writing the code myself forces me to think through every pointer, every recursion, every edge case, and every design decision instead of simply reading someone else's implementation.

If my explanations or implementations happen to help another student taking COS212 (or anyone learning DSA), then that's an added bonus.

## Goals

* Build every data structure from scratch in Java.
* Develop a deeper intuition for algorithms instead of memorising them.
* Document what I learn along the way.
* Create a resource that other students can learn from.
* Improve my problem-solving and software engineering skills.

If you spot something that can be improved, feel free to open an issue or submit a pull request. I'm here to learn.
